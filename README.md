# Centralized Log Management System

## Overview

A professional-grade, MVC-based PHP application for collecting, analyzing, and managing application logs with advanced features like pattern detection, search & filtering, and exportable reports.

## Features

✅ **Severity Levels** - DEBUG, INFO, WARNING, ERROR, CRITICAL
✅ **Pattern Detection** - Automatic anomaly detection based on regex patterns
✅ **Advanced Search** - Filter by severity, source, message, date range, and tags
✅ **Exportable Reports** - JSON, CSV, HTML formats
✅ **REST API** - Full REST API for log operations
✅ **Dashboard** - Real-time statistics and visualizations
✅ **Dark Mode** - Toggle between light and dark themes
✅ **Responsive Design** - Mobile-friendly interface
